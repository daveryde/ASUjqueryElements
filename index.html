What Is A jQuery Selector?

In every jQuery statement, the $ is followed by a reference to the element or elements on which the statement operates, contained in parentheses. To select elements in jQuery, you specify a CSS selector, contained in single or double quotes. For instance, to start a jQuery statement that operates on li elements, you use the following syntax.

jQuery Selector Example

$(document).ready(function () {
  /* Find li element */
  var listElement = $('li');
}

jQuery Selector In Action

Creating elements can be accomplished with the same selector syntax that we used to find elements. The selector will traverse the DOM looking for what you place inside the selector or create an element if you explicity place an HTML tag element in the selector. This way, jQuery element functions will work on this dynamically created element before being added to the DOM.
